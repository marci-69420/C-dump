# C Dump

This repository contains C programming exercises and assignments completed during my studies at LUT University. It serves as a collection of practical implementations covering various fundamental and advanced topics in the C programming language.

## Topics Covered

- **Structure and Control Flow** - Basic control structures and program flow
- **Handling Data and IO in C** - Input/output operations and data handling
- **Functions** - Function design and implementation
- **Dynamic Memory Allocation** - Memory management and dynamic structures
- **Data Structures and Advanced Memory Management** - Complex data structures and memory techniques
- **Structuring Data** - Structs and data organization
- **Advanced Pointer Topics** - Pointer arithmetic and advanced pointer concepts
- **Exploring C Header Files** - Working with standard library headers
- **Working with Binary Data** - Binary operations and file handling
